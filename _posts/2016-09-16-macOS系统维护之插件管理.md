* 推荐使用homebrew管理内部插件，brewcask下载更新app

* 下载homebrew后可以管理下载ruby-面对对象的脚本语言

  ```shell
  $ brew install ruby
  ```

* Gem是ruby的第三方插件管理器

  ```shell
  ## 检查gem版本
  gem -v
  ## 更新Gem(提示权限) sudo
  gem update
  gem update --system 
  ```

  gem update --system。这一步需要翻墙，否则会出现404错误。解决办法参考：https://ruby.taobao.org/
